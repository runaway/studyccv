Library Reference

  • data structure utilities: lib/ccv_util.c
  • image transform utilities: lib/ccv_transform.c
  • track learn detect: lib/ccv_tld.c
  • stroke width transform: lib/ccv_swt.c
  • scale invariant feature transform: lib/ccv_sift.c
  • HTTP server: serve/*.c
  • image resampling utilities: lib/ccv_resample.c
  • numerical algorithms: lib/ccv_numeric.c
  • memory alloc/dealloc: lib/ccv_memory.c
  • basic IO utilities: lib/ccv_io.c
  • integral channel features: lib/ccv_icf.c
  • deformable parts model: lib/ccv_dpm.c
  • DAISY: lib/ccv_daisy.c
  • Deep Convolutional Networks: lib/ccv_convnet.c
  • classic computer vision algorithms: lib/ccv_classic.c
  • cache mechanism: lib/ccv_cache.c
  • binary brightness feature: lib/ccv_bbf.c
  • basic image processing utilities: lib/ccv_basic.c
  • linear algebra: lib/ccv_algebra.c
